1. html 템플릿 다운받기

이번에 html을 처음 써보기도 하고, 시간도 부족해서 상업적으로 무료인 템플릿을 사용하기로 했습니다.

저는 인터넷에서 Tooplate란 사이트에서 만든 'LITTLE FASHION FREE CSS TEMPLATE'을 다운받았습니다.

2. 나만의 웹사이트 만들기 시작

다운로드를 받으면 다음과 같은 파일이 생기는데, 여기서 사이트 html 파일인 index.html을 텍스트 에디터인 Visual Studio Code로 열어줍니다.

그럼 이제 이 사이트는 제 것입니다. 제 마음대로 바꾸고 팔(?) 수 있는겁니다.

3. 웹 개발하기

혼동하는 사람들이 많아서 그런데 html은 프로그램 언어가 아니라 마크업 언어입니다.

쉽게 말해서 한컴오피스 한글로 문서를 예쁘게 꾸민다고 해서 그걸 코딩이라고 부르진 않잖아요?


잡설은 각설하고, 아까 불러와준 파일에서 필요한 부분만 남기고 제거해줍니다.

그러면서 필요한 소스들(ex 이미지, 영상, 글씨)을 새로 넣거나 교체해주고, 이들의 간격과 위치를 보기 좋게 재조정해줍니다.

디자이너분이 미리 연결해둔 JAVASCRIPT도 잘 활용해줍니다. ex) 애니메이션 등등

4. 완료..?

아무래도 초보자이다보니 노가다도 많이 하고, 코드가 좀 더러워졌습니다.

그래도 구동은 제대로 하니까 어영부영 완성은 되었네요. 뿌듯합니다.

코드가 너무 많고 길어서 사진을 전부 다 첨부하지 못한 점 참고바랍니다. 혹시라도 필요하시면 연락주세요(최하단 참조).

5. 웹 호스팅

...그러나 아직 컴퓨터를 덮을 순 없습니다. 아직 웹 호스팅이란 숙제가 남아있습니다.

지금까지 만든 사이트는 공유/로컬 파일로만 열리기 때문에 원래 목적이었던 접근성 향상과는 부합하지 못합니다.

(웹페이지 상단에 웹주소 좌측을 보면 '파일'이라고 적힌 것을 확인할 수 있습니다.)


'호스트'는 인터넷으로 연결된 모든 디바이스를 뜻하고, '호스팅'은 호스트를 빌려주는 행위를 뜻합니다.

즉, '웹 호스팅'이란 개별 홈페이지를 운영하는 사용자를 위해 서버 컴퓨터의 일부 공간을 임대해 주는 서비스를 말합니다.

웹 페이지를 배포하기 위해서는 서버 프로그램을 실행시켜야합니다. 하지만 이를 개인이 하는 건 무리이기 때문에 호스팅 해주는 업체를 빌려야합니다.

저는 netflify를 쓰기로 했습니다. 무료거든요.

6. 진짜 끝

드디어! 마참내! 이제 진짜 끝끝끝 The End입니다.

인터넷에 연결된 누구든, 당신이 보고 있는 이 사이트에 접속할 수 있게 된 것입니다! 여러부운!
