#세세한 프로그래밍 과정

#약한(?) 스압주의

요약: 원래 C++ 언어로 프로그래밍하다 컴퓨터 문제가 생겨서 Python으로 바꿨고, 결과까지 잘 나왔으나 공유시의 문제가 생겼습니다.

프로그램 제작과정 사진

1. 개발 언어 C++을 실행

저는 DEV C++이란 어플을 사용했습니다.

2. 코딩하기

C++로 컴퓨터가 필요한 값을 계산 할 수 있도록 구현해줍니다.

3. C++ 코드 실행

결과가 잘 나오는군요. 그런데 뭔가 아쉽습니다.

4. 프로그램의 인터페이스 만들기

GUI(Graphical User Interface)라고 부릅니다.

아무래도 단순히 결과값이 나열되는 것보단 사용자 친화적이게끔 직접 실행할 수 있게 하는게 더욱 낫겠죠.

따라서 저는 C++의 GUI를 디자인할 수 있는 QT Creator/Designer란 프로그램을 사용하려 했습니다.. 했으나..

5. 파이썬으로 코딩 다시 짜기

제 컴퓨터 시스템 상 내부에서 충돌이 일어나 설치가 계속 안되더군요..ㅠㅠ

눈물을 머금고 C++ 대신 Python을 사용하여 코드를 다시 제작합니다.

그러면서 덤으로 부족했던 부분들을 보충해줍니다.

6. Python 코드 실행

결과가 잘 나오는군요. 그리고 역시나 GUI가 아쉽습니다.

7. Tkinter로 GUI 만들기

파이썬은 GUI를 나타내는 내장함수 Tkinter가 이미 있어 다행입니다.

앞써 제작한 파이썬 코드를 Tkinter 모듈에 있는 기능들을 이용해 다시 조립해줍니다.

그러면서 이번에도 좀 더 편리한 UI를 만들어보고자 필요한 부분들을 추가해줍니다.

처음해보는 거라 많이 힘들었던 건 덤.. 저 코드 왼편에 나열된 수많은 시행착오들을 보십쇼..흑흑

8. 개편한 Python 코드 실행

여러번 실행해서 문제가 없는지 확인해봅니다. 제대로 잘 나오네요...

9. 완료..?

이리하여 코드 '1-newnew-3-2.py'는 파이썬이 설치된 컴퓨터에서 잘 작동하게 되었습니다

-만 이러면 접근성이 너무 떨어지잖아요?

그러니 파이썬이 없어도 모든 컴퓨터에서 실행되게끔 만들어봅시다.

다만 제가 작업한 시스템이 Windows OS여서 맥북에선 실행안되는 게 함정 (^人^)

10. 확장자 바꾸기

방법은 간단합니다. 기존의 .py에서 .exe로 변환해주면돼요.

파이썬에서 pyinstaller 모듈을 사용해 쉽게 만들 수 있습니다.

현재 안타깝게도 제 컴퓨터에선 pyinstaller가 계속해서 오류를 내는 바람에 파일 자체를 공유하기 어려운 상황입니다..
그래도 꼭 빠른 시간 안에 '20821채서연-program.exe'란 이름으로 돌아오겠습니다!
